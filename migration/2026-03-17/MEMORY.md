## 📖 使用说明

### 写入记忆
- **项目相关** → `memory/projects/<name>.md`
- **日常对话** → `memory/daily/YYYY-MM-DD.md`
- **配置变更** → `memory/config/<category>.md`
- **知识学习** → `memory/knowledge/<topic>.md`
- **重要决策** → `memory/decisions/YYYY-MM-DD-topic.md`

### 审查周期
- **daily/**: 每 7 天
- **projects/**: 项目阶段结束
- **knowledge/**: 每月
- **decisions/**: 每季度
