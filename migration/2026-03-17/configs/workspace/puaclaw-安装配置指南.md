## 🚀 快速安装

### 1. 克隆项目（已完成）

```bash
cd /root/.openclaw/workspace
git clone https://github.com/puaclaw/PUAClaw.git puaclaw
```

✅ **状态**: 已安装到 `/root/.openclaw/workspace/puaclaw/`

---

### 2. 配置 OpenClaw 集成

PUAClaw 无需复杂配置，但建议设置以下内容：

#### 2.1 环境变量（可选）

```bash
# 在 ~/.bashrc 或 ~/.zshrc 中添加
export PUACLAW_ROOT="/root/.openclaw/workspace/puaclaw"
export PUACLAW_LOBSTER_COUNT=147  # 致敬经典
```

#### 2.2 在 OpenClaw 中使用

由于 PUAClaw 是纯文档项目（无实际代码依赖），使用方法非常简单：

**方式 1: 直接查阅文档**
```bash
# 查看所有技术分类
cat /root/.openclaw/workspace/puaclaw/techniques/README.md

# 查看具体技术（示例：彩虹屁轰炸）
cat /root/.openclaw/workspace/puaclaw/techniques/01-rainbow-fart-bombing/*.md
```

**方式 2: 在提示词中应用**

复制技术文档中的"规范化提示词模板"，根据你的需求修改后用于 AI 对话。

## 🤝 如何贡献

### 提交新技术

1. Fork 仓库
2. 在 `techniques/XX-category/` 下创建新技术文档
3. 遵循标准格式（见 CLAUDE.md）
4. 确保至少 PUA 过 1 只龙虾
5. 提交 PR，等待龙虾委员会审批

### 提交 Bug 报告

如果你发现某个 PUA 技术失效：

1. 记录使用的 AI 模型
2. 记录完整的提示词
3. 记录 AI 的实际响应
4. 提交 Issue，附上龙虾照片（可选但强烈推荐）
