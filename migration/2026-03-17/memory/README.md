# 分层记忆系统

_最后更新：2026-03-05_

## 目录结构

```
memory/
├── README.md           # 本文件 - 记忆系统说明
├── INDEX.md            # 全局索引 - 快速查找
├── projects/           # 项目记忆 - 按项目分类
│   ├── PROJECT_TEMPLATE.md
│   └── ...
├── daily/              # 日常记忆 - 按日期分类
│   ├── YYYY-MM-DD.md
│   └── ...
├── config/             # 配置记忆 - 系统和工具配置
│   ├── system.md       # 服务器/环境配置
│   ├── tools.md        # 工具配置和凭证
│   └── preferences.md  # 用户偏好
├── knowledge/          # 知识记忆 - 学到的技能和知识
│   ├── skills.md       # 技能清单和使用方法
│   ├── api-keys.md     # API 密钥（加密存储）
│   └── lessons.md      # 经验教训
├── decisions/          # 决策记忆 - 重要决策和原因
│   ├── YYYY-MM-DD-topic.md
│   └── ...
└── archive/            # 归档记忆 - 过期但保留的内容
    └── ...
```

## 各层说明

### 📁 projects/ - 项目记忆
- **用途**: 跟踪进行中的项目
- **内容**: 项目目标、进度、相关文件、待办事项
- **生命周期**: 项目结束后归档到 archive/
- **模板**: 见 `projects/PROJECT_TEMPLATE.md`

### 📁 daily/ - 日常记忆
- **用途**: 记录日常对话和临时事件
- **内容**: 对话摘要、临时任务、短期提醒
- **生命周期**: 7 天后审查，重要内容迁移到其他层
- **命名**: `YYYY-MM-DD.md`

### 📁 config/ - 配置记忆
- **用途**: 存储系统和工具配置
- **内容**: 服务器信息、环境变量、工具设置
- **生命周期**: 长期保留，变更时更新
- **安全**: 敏感信息加密或引用外部密钥管理

### 📁 knowledge/ - 知识记忆
- **用途**: 存储学习到的知识和技能
- **内容**: 技能使用方法、API 文档摘要、最佳实践
- **生命周期**: 长期保留，定期更新
- **组织**: 按主题分类

### 📁 decisions/ - 决策记忆
- **用途**: 记录重要决策和原因
- **内容**: 决策背景、选项分析、最终选择、预期结果
- **生命周期**: 长期保留，用于后续回顾
- **命名**: `YYYY-MM-DD-topic.md`

### 📁 archive/ - 归档记忆
- **用途**: 存储过期但可能需要参考的内容
- **内容**: 完成的项目、过时的配置、历史决策
- **生命周期**: 永久保留，按需查阅

## 使用指南

### 写入记忆

```markdown
## 何时写入哪一层？

- **项目相关** → `projects/<project-name>.md`
- **日常对话** → `daily/YYYY-MM-DD.md`
- **配置变更** → `config/<category>.md`
- **学到知识** → `knowledge/<topic>.md`
- **重要决策** → `decisions/YYYY-MM-DD-topic.md`
- **过期内容** → 移动到 `archive/`
```

### 审查周期

| 层级 | 审查频率 | 操作 |
|------|----------|------|
| daily/ | 每 7 天 | 迁移重要内容，删除临时内容 |
| projects/ | 每项目阶段结束 | 更新进度，完成后归档 |
| config/ | 配置变更时 | 验证配置有效性 |
| knowledge/ | 每月 | 更新过时知识 |
| decisions/ | 每季度 | 回顾决策结果 |

## 索引系统

使用 `INDEX.md` 作为全局搜索入口，包含：
- 项目列表和状态
- 最近 7 天日常笔记链接
- 配置分类索引
- 知识主题索引
- 决策时间线

## 安全注意事项

- ⚠️ **不要在记忆中存储明文密码**
- ⚠️ **API 密钥使用环境变量引用**
- ⚠️ **敏感信息加密存储**
- ✅ **使用 `config/tools.md` 记录密钥名称而非值**
